---
title: Delete Memory
api: DELETE https://api.remem.online/memories/{memory_id}
description: Delete a single memory by ID.
---

## Overview

Surgically remove one specific memory. The `user_id` and `agent_id` must match the memory's owner — you can't delete another user's memories.

---

## Request

### Headers

| Name | Required | Description |
|------|----------|-------------|
| `X-API-Key` | Yes | Your Remem API key |

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `memory_id` | UUID of the memory to delete. |

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `user_id` | string | Yes | Must match the memory owner. |
| `agent_id` | string | Yes | Must match the memory agent. |

<CodeGroup>
```bash curl
curl -X DELETE \
  "https://api.remem.online/memories/775263ee-73af-4416-9804-1f274048ae08?\
user_id=user_123&agent_id=support_bot" \
  -H "X-API-Key: rm_live_xxx"
```

```python Python SDK
result = client.forget(
    memory_id="775263ee-73af-4416-9804-1f274048ae08",
    user_id="user_123",
    agent_id="support_bot",
)
print(result.deleted)  # 1
```
</CodeGroup>

---

## Response

### 200 — Success

```json
{
  "deleted": 1,
  "message": "Deleted 1 memory."
}
```

### Memory Not Found

```json
{
  "deleted": 0,
  "message": "Memory not found."
}
```

Returns `deleted: 0` if the ID doesn't exist or belongs to a different user. Does not return an error — safe to call even if unsure whether the memory exists.
