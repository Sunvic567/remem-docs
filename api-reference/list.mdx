---
title: List Memories
api: GET https://api.remem.online/memories
description: Browse all stored memories for a user and agent pair.
---

## Overview

Returns all memories for a user and agent, sorted by most recent first. Useful for building a memory browser in your dashboard, debugging what your agent knows, or auditing stored data.

---

## Request

### Headers

| Name | Required | Description |
|------|----------|-------------|
| `X-API-Key` | Yes | Your Remem API key |

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `user_id` | string | Yes | — | List memories for this user. |
| `agent_id` | string | Yes | — | List memories for this agent. |
| `memory_type` | string | No | `null` | Filter by `episodic`, `semantic`, or `summary`. |
| `limit` | integer | No | `20` | Results per page. Min 1, max 100. |
| `offset` | integer | No | `0` | Pagination offset. |

<CodeGroup>
```bash curl
curl "https://api.remem.online/memories?\
user_id=user_123&agent_id=support_bot&limit=20&offset=0" \
  -H "X-API-Key: rm_live_xxx"
```

```python Python SDK
# First page
page = client.list(
    user_id="user_123",
    agent_id="support_bot",
    limit=20,
    offset=0,
)

print(f"Total memories: {page.total}")
for m in page.memories:
    print(m.content, m.created_at)

# Next page
page2 = client.list(
    user_id="user_123",
    agent_id="support_bot",
    limit=20,
    offset=20,
)
```
</CodeGroup>

## Response

### 200 — Success

```json
{
  "memories": [...],
  "total":  47,
  "limit":  20,
  "offset": 0
}
```

| Field | Type | Description |
|-------|------|-------------|
| `total` | integer | Total number of memories for this user+agent |
| `limit` | integer | Number returned in this response |
| `offset` | integer | Current pagination offset |
